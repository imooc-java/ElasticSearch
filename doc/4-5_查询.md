## 初始化 book

### 创建索引

PUT http://127.0.0.1:9200/book

* 请求

```json
{
  "mappings": {
    "novel": {
      "properties": {
        "word_count": {
          "type": "integer"
        },
        "author": {
          "type": "keyword"
        },
        "title": {
          "type": "text"
        },
        "publish_date":{
          "type": "date",
          "format": "yyyy-MM-dd HH:mm:ss||yyyy-MM-dd||epoch_millis"
        }
      }
    }
  }
}
```

* 返回

```json
{
    "acknowledged": true,
    "shards_acknowledged": true,
    "index": "book"
}
```

## 简单查询

### 根据id查询

GET http://127.0.0.1:9200/book/novel/1

* 请求

  无

* 返回

```json
{
    "_index": "book",
    "_type": "novel",
    "_id": "1",
    "_version": 1,
    "found": true,
    "_source": {
        "author": "王五",
        "title": "菜谱",
        "word_count": 5000,
        "publish_date": "2002-10-01"
    }
}
```
